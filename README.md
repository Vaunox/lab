# The Lab

An open-source research kernel for systematic strategies on Indian cash equity.

**It ships the machine, not the cartridges.** No strategies. No signals. No data.
No path to an exchange — not now, not later.

Its product is a verdict you can trust, with the trial count and the haircut
stapled to it.

> **This is not an AI that finds strategies. It is an AI that will mostly tell you
> no, very quickly.**
